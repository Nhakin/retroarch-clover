=== PSSX ReARMed core for RetroArch ===

This module will add support for PlayStation games to RetroArch
The following BIOS images are required for this core to work properly:
scph5500.bin
scph5501.bin
scph5502.bin

Available executables and arguments:
- /bin/pcsx <rom> <clover_args>
  runs "pcsx_rearmed" core
