=== Cave Story (NX Engine) core for RetroArch ===

This module will add support for Cave Story Game Engine games to RetroArch
The following data ROM is required for this core to work properly:
Cave Story (en).zip

Available executables and arguments:
- /bin/nxengine <rom> <clover_args>
  runs "nxengine" core
